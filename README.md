# vim-commands
A helpful list of commands for faster work with vim.

| Path      | Key | 
| :---        |    :----:   |
| Move left    |  `h`     |
| Move right    |  `j`     |
| Move up    |  `k`     |
| Move down    |  `l`     |
| Move to a specific line    |  `#G / #gg`     |
| Insert mode    |  `i`     |
| Exit insert mode    |  `Esc`     |
| Save the file    |  `Esc + :w`     |
| Save the file    |  `Esc + :w`     |
| Save & close the file    |  `Esc + :x`     |
| Quit    |  `Esc + :q`     |
